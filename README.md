## build
```bash
$ git https://github.com/michael199619/IponwebHackathon2021
$ cd IponwebHackathon2021
$ cp .env.example .env
$ yarn install
```

## start

```bash
$ node build/main.js -h 500 -w 500 example/IMG_0603.HEIC // change size of image
$ node build/main.js -c example/IMG_0603.HEIC // city
```

## tests
```bash
$ yarn test
```
